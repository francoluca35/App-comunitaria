# Plataforma de Difusión Comunitaria

Aplicación mobile-first para difusión de información comunitaria con sistema de moderación.

## 🚀 Características

### Para Usuarios Comunitarios
- ✅ Registro e inicio de sesión (email/contraseña y Google)
- ✅ Crear publicaciones con imágenes y categorías
- ✅ Ver feed de publicaciones aprobadas
- ✅ Filtrar por categorías (Mascotas, Alertas, Avisos, Objetos, Noticias)
- ✅ Comentar publicaciones
- ✅ Contactar autores por WhatsApp
- ✅ Modo claro/oscuro

### Para Administradores
- ✅ Dashboard con estadísticas
- ✅ Moderar publicaciones pendientes
- ✅ Aprobar/rechazar publicaciones completas
- ✅ Rechazar imágenes individuales
- ✅ Gestionar usuarios (bloquear/desbloquear)
- ✅ Configurar límites y funcionalidades
- ✅ Gestionar términos de servicio

## 📱 Navegación

La app usa bottom navigation con:
- **Inicio**: Feed de publicaciones
- **Crear**: Nueva publicación
- **Panel**: Dashboard de admin (solo para administradores)
- **Perfil**: Información del usuario

## 🎨 Diseño

- Mobile-first responsive
- Modo claro/oscuro
- Estética moderna y minimalista
- Colores que transmiten confianza y seguridad

## 📋 Estados de Publicaciones

1. **Pendiente** (🟡): Esperando moderación
2. **Aprobada** (🟢): Visible públicamente
3. **Rechazada** (🔴): No aprobada por moderación

## ⚙️ Configuración (Admin)

- Activar/desactivar comentarios
- Activar/desactivar contacto por WhatsApp
- Límite de publicaciones por usuario
- Límite de imágenes por publicación
- Términos y condiciones personalizables
